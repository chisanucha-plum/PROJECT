# Helmet Compliance and Traffic Violation Detection System  

## 📌 Overview  
This project is a **Helmet Compliance and Traffic Violation Detection System** that uses **YOLOv8** for real-time object detection. It identifies whether a motorcyclist is wearing a helmet and detects **traffic violations** such as triple riding. The system is designed to improve road safety within **university campuses** or other controlled environments.  

## 🚀 Features  
- **Helmet Detection**: Identifies whether a motorcyclist is wearing a helmet.  
- **Triple Riding Detection**: Detects if more than two people are riding a single motorcycle.  
- **Real-time Processing**: Uses **OpenCV** and **YOLOv8** for efficient frame-by-frame detection.  
- **Flask API**: Integrates a web-based API to process video/image inputs.  
- **OCR & BERT Integration**: Extracts text and applies AI-based text correction.  

## 🛠️ Technologies Used  
- **Python** (Flask, OpenCV, NumPy, Matplotlib)  
- **YOLOv8** (Ultralytics for object detection)  
- **Tesseract OCR** (for text extraction)  
- **BERT/DistilBERT** (for text processing and correction)  
- **MongoDB** (for data storage)
